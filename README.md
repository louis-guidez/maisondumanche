# La Maison du Manche - Design System

Ce repository contient le **design system en cours de construction** pour la marque **La maison du manche**.

## Contexte de marque

- Nom de la marque : **La maison du manche**
- Slogan : **Un manche, mille usages.**
- Produit : **Grand set de manches avec son lot d'outils interchangeables**

## Objectif du projet

Construire un design system React réutilisable (atoms, molecules, components), avec des pages de démonstration, pour poser des bases UI cohérentes avant industrialisation.

## État actuel

- Monorepo avec :
  - `apps/web` : application de démonstration
  - `packages/designsystem` : bibliothèque de composants
- Architecture design system en place :
  - `atoms`
  - `molecules`
  - `components`
- Pages principales existantes :
  - Contact
  - Articles
  - Tous les composants

## Lancer le projet

```bash
npm install
npm run dev
```

## Build

npm run build
